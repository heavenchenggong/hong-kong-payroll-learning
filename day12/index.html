📜 一、终止雇佣法律法规基础

1.1 法律依据

《雇佣条例》（第32章）（Employment Ordinance, Cap. 32）- 香港雇佣关系的法律基础，涵盖终止雇佣、遣散费、长期服务金等关键条款。

1.2 终止雇佣的定义

🎯 什么是终止雇佣？

终止雇佣是指雇主和雇员之间的雇佣关系结束。这可能由多种原因引起：

  • 雇主主动终止：雇主因业务需要或雇员原因终止雇佣
  • 雇员主动终止：雇员因个人原因辞职
  • 双方协商终止：雇主和雇员达成协议终止雇佣
  • 雇佣到期：固定期限合同到期
  • 雇员死亡：雇员去世导致雇佣关系终止

1.3 适用范围

类别 遣散费适用 长期服务金适用 说明
连续性雇佣员工 ✅ 是 ✅ 是 所有连续性雇佣员工
固定期限合同 ✅ 是 ✅ 是 符合资格条件
临时工 ❌ 否 ❌ 否 非连续性雇佣
公务员 ❌ 否 ❌ 否 由政府条例管辖

📅 二、终止通知期

2.1 通知期规定

服务年资 雇主通知期 雇员通知期
不满1个月 无通知期 无通知期
1个月至不满2年 1个月 1个月
2年至不满5年 1个月 1个月
5年至不满10年 2个月 1个月
10年或以上 2个月 1个月

2.2 代通知金

💰 代通知金计算

  • 定义：未按规定通知期通知对方而终止雇佣时，需要支付的补偿
  • 计算方式：代通知金 = 月收入 x 通知月数
  • 月收入计算：最近30天的工资 / 30（包括基本工资、津贴、佣金等）
  • 上限：代通知金上限为 HKD 22,500

2.3 无需通知的情况

⚠️ 可立即终止雇佣的情况

  • 合理理由解雇：雇员有严重违反公司规定或行为不当
  • 雇员严重不当行为：如欺诈、盗窃、严重疏忽等
  • 试用期解雇：在试用期内，雇主可提前7天通知终止
  • 裁员：雇主因业务需要裁员时，需支付遣散费

💰 三、遣散费（Severance Payment）

3.1 遣散费定义

遣散费是指雇主因裁员而终止雇佣时，必须向符合条件的雇员支付的补偿。

3.2 遣散费资格条件

✅ 遣散费资格要求

  • 服务年资：雇员必须连续服务不少于24个月
  • 终止原因：因裁员而被解雇（即雇主因业务需要而裁减员工）
  • 非自愿终止：非因雇员不当行为而终止
  • 非退休：非因退休而终止雇佣

3.3 遣散费计算公式

🧮 遣散费计算公式

遣散费 = 2/3 x 月收入 x 服务年资

  • 2/3：固定比例系数
  • 月收入：最近12个月的平均月收入（上限 HKD 22,500）
  • 服务年资：连续服务年数，上限为12年
  • 最高限额：遣散费上限为 HKD 390,000

示例：服务10年，月收入 HKD 18,000

遣散费 = 2/3 x 18,000 x 10 = HKD 120,000

3.4 遣散费扣减

📋 遣散费扣减情况

  • MPF供款：遣散费可扣减雇主在雇员离职前已支付的MPF供款
  • 其他补偿：如同时符合长期服务金条件，只支付较高的一项
  • 已收补偿：雇主已支付的代通知金可从遣散费中扣减

🏆 四、长期服务金（Long Service Payment）

4.1 长期服务金定义

长期服务金是指雇员因雇主未能续约而终止雇佣，或因退休而终止雇佣时，应得的补偿。

4.2 长期服务金资格条件

✅ 长期服务金资格要求

  • 服务年资：连续服务不少于24个月（固定期限合同）或5年（连续性雇佣）
  • 终止原因：因雇主未能续约或雇员退休而终止
  • 非裁员：非因裁员而终止（裁员应获遣散费）
  • 非不当行为：非因雇员严重不当行为而终止

4.3 长期服务金计算公式

🧮 长期服务金计算公式

长期服务金 = 2/3 x 月收入 x 服务年资

  • 计算方法：与遣散费相同
  • 月收入上限：HKD 22,500
  • 服务年资上限：12年（长期服务金无上限）
  • 最高限额：长期服务金上限为 HKD 390,000

示例：服务15年，月收入 HKD 20,000

长期服务金 = 2/3 x 20,000 x 12 = HKD 160,000

注：服务年资按12年计算，因有年资上限

4.3 遣散费与长期服务金的区别

项目 遣散费 长期服务金
支付原因 裁员 雇主未能续约或退休
服务年资 最少24个月 固定合同：24个月；连续雇佣：5年
年资上限 12年 无上限
计算公式 相同 相同
可否同时获得 ❌ 否 ❌ 否

🔄 五、终止雇佣处理流程

5.1 雇主主动终止流程

📝 终止步骤

  1. 评估终止原因：确认终止雇佣的合法性和合规性
  2. 确定补偿方案：根据服务年资和终止原因计算遣散费/长期服务金
  3. 发出通知：按规定通知期向雇员发出终止通知
  4. 计算未付工资：结算未付工资、假期补偿、佣金等
  5. 计算代通知金（如适用）
  6. 安排最后工作日：确定最后工作日期
  7. 支付补偿：在最后工作日或之前支付所有补偿
  8. 完成离职手续：退还公司财物、签署离职文件
  9. 更新人事记录：在系统中更新雇员状态

5.2 雇员主动终止流程

📝 辞职步骤

  1. 提交辞职信：按规定通知期提交辞职申请
  2. 确认最后工作日：与雇主确认最后工作日期
  3. 交接工作：完成工作交接和文档移交
  4. 退还公司财物：归还所有公司财产
  5. 结算未付补偿：雇主结算未付工资、年假补偿等
  6. 签署离职文件：签署离职证明和确认书
  7. 更新人事记录：系统更新雇员状态

5.3 付款要求

项目 支付时限 说明
未付工资 最后工作日或之前 包括应发工资和津贴
年假补偿 最后工作日或之前 按比例折算的年假补偿
遣散费/长期服务金 最后工作日或之前 按法定标准支付
代通知金 最后工作日或之前 如未按规定通知期通知

💵 六、其他补偿与注意事项

6.1 年假补偿

🏖️ 年假补偿计算

  • 未休年假：雇员离职时未休的年假必须折算成现金补偿
  • 计算方法：年假补偿 = 日薪 x 未休年假天数
  • 日薪计算：月收入 x 12 / 365（不含超时工作费）
  • 注意事项：年假补偿必须在最后工作日或之前支付

6.2 期末支付

💰 期末支付项目

  • 工资：至最后工作日的应发工资
  • 津贴：住房津贴、交通津贴等未付津贴
  • 佣金：应发而未发的佣金
  • 加班费：应发而未发的加班费
  • 奖金：如适用于未服务月份

6.3 注意事项

⚠️ 雇主必须注意

  • 合理理由解雇：如无合理理由解雇，雇员可向劳资审裁处申诉
  • 歧视性解雇：不得因种族、性别、怀孕等原因解雇
  • 工会活动解雇：不得因雇员参与工会活动而解雇
  • 职业病解雇：雇员因工伤或职业病暂时丧失工作能力时不得解雇
  • 怀孕解雇：雇员怀孕期间不得解雇（严重不当行为除外）

⚙️ 七、SAP系统中的终止雇佣处理

7.1 SAP HCM中的终止雇佣操作

🔧 关键操作步骤

  1. 进入终止雇佣操作：T-Code PA40（人事动作）
  2. 选择动作类型：选择"终止雇佣"（Termination）
  3. 输入终止日期：确定雇佣终止的生效日期
  4. 选择终止原因：选择合适的终止原因代码（如裁员、辞职、退休）
  5. 维护信息类型：系统自动更新相关IT（如IT0000人事事件）
  6. 计算补偿：系统根据规则计算遣散费、长期服务金等
  7. 生成工资单：运行离职工资单，计算期末支付
  8. 归档数据：系统归档雇员数据

7.2 关键信息类型

信息类型 用途 关键字段
IT0000 人事事件 终止日期、终止原因、动作类型
IT0014 经常性支付/扣除 遣散费、长期服务金、代通知金
IT0008 基本工资 用于计算补偿的月收入
IT0015 额外支付 期末支付、年假补偿
IT0026 年假缺勤 未休年假天数

7.3 工资核算配置

💰 工资核算规则

  • 遣散费计算：通过工资核算规则自动计算遣散费
  • 长期服务金计算：根据服务年资和终止原因自动计算
  • 年假补偿计算：系统自动计算未休年假的补偿金额
  • 期末支付：结算所有未付的工资、津贴、佣金等
  • 税项计算：遣散费和长期服务金是否需要缴税视具体情况

7.4 香港Payroll特有配置

🇭🇰 香港Payroll配置要点

  • 离职原因代码：维护符合香港法规的终止原因代码
  • 补偿规则：配置遣散费和长期服务金的计算规则
  • 上限控制：设置遣散费和长期服务金的上限（HKD 390,000）
  • 通知期规则：根据服务年资配置通知期
  • MPF扣减：配置MPF供款从遣散费中扣减的规则

📊 八、实践案例分析

案例1：裁员遣散费计算

背景：某公司因业务重组需裁员一名员工。

员工信息：

  • 服务年资：8年零6个月
  • 月收入：HKD 20,000
  • 终止原因：裁员

遣散费计算：

  • 服务年资：8年（不满1年按1年计算）
  • 遣散费 = 2/3 x 20,000 x 8 = HKD 106,667
  • 代通知金：雇主提前1个月通知，无需支付代通知金
  • 未休年假：5天，补偿 = 20,000 x 12 / 365 x 5 = HKD 3,288
  • 期末支付：工资支付至最后工作日

总补偿：HKD 106,667 + HKD 3,288 = HKD 109,955

案例2：长期服务金计算

背景：一名员工在公司服务12年后，公司决定不再续约。

员工信息：

  • 服务年资：12年零3个月
  • 月收入：HKD 25,000（超过上限 HKD 22,500）
  • 终止原因：雇主未能续约

长期服务金计算：

  • 月收入：按上限计算 = HKD 22,500
  • 服务年资：12年（超过12年按12年计算）
  • 长期服务金 = 2/3 x 22,500 x 12 = HKD 180,000
  • 未休年假：10天，补偿 = 22,500 x 12 / 365 x 10 = HKD 7,397

总补偿：HKD 180,000 + HKD 7,397 = HKD 187,397

案例3：代通知金计算

背景：雇主因严重不当行为解雇一名服务6年的员工。

员工信息：

  • 服务年资：6年
  • 月收入：HKD 18,000
  • 终止原因：严重不当行为（合理理由解雇）
  • 通知期：雇主未提前通知，立即解雇

补偿计算：

  • 通知期：服务6年，雇主应提前2个月通知
  • 代通知金 = 18,000 x 2 = HKD 36,000
  • 遣散费/长期服务金：因严重不当行为解雇，无需支付
  • 未休年假：3天，补偿 = 18,000 x 12 / 365 x 3 = HKD 1,773

总补偿：HKD 36,000 + HKD 1,773 = HKD 37,773

关键点：合理理由解雇无需支付遣散费，但仍需支付代通知金（如未按规定通知）。

✨ 九、终止雇佣管理最佳实践

9.1 预防性措施

🛡️ 雇佣管理建议

  • 明确雇佣合同：在合同中明确雇佣条款和终止条件
  • 定期评估：定期评估员工表现，及早发现和解决问题
  • 建立政策：制定明确的终止雇佣政策和流程
  • 培训管理人员：确保管理人员了解雇佣法规和最佳实践

9.2 人力资源管理系统化

📋 系统化管理

  • 自动化计算：使用SAP系统自动计算遣散费和长期服务金
  • 合规检查：系统自动检查终止雇佣的合规性
  • 审计追踪：保留完整的终止雇佣记录和审计线索
  • 报告生成：生成终止雇佣报告，用于管理决策

9.3 法律风险防范

⚠️ 法律风险防范

  • 避免歧视性解雇：确保终止雇佣不违反反歧视法规
  • 合理理由解雇：确保解雇有充分的法律依据和证据
  • 咨询法律意见：在复杂情况下咨询专业律师
  • 保存证据：保存所有相关文档和证据，以备日后需要

📝 十、学习总结

🎯 今日核心要点

  • 终止雇佣由《雇佣条例》（第32章）管辖
  • 终止通知期根据服务年资从7天到2个月不等
  • 遣散费适用于裁员情况，计算公式：2/3 x 月收入 x 服务年资
  • 长期服务金适用于雇主未能续约或退休，计算公式相同
  • 遣散费和长期服务金上限为 HKD 390,000
  • 代通知金为月收入 x 通知月数，上限为 HKD 22,500
  • SAP系统通过T-Code PA40处理终止雇佣
  • 关键信息类型包括IT0000（人事事件）、IT0014（补偿支付）等
