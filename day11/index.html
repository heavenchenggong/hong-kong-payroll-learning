📜 一、ECI法律法规基础

1.1 法律依据

《雇员补偿条例》（第282章）（Employees' Compensation Ordinance, Cap. 282）- 香港雇员补偿保险制度的法律基础。

1.2 立法目的

🎯 立法目标

  • 保障雇员：为因工受伤或患职业病的雇员提供经济补偿
  • 转移风险：通过保险机制转移雇主的财务风险
  • 促进安全：激励雇主改善工作环境，减少工伤事故

1.3 适用范围

类别 说明 是否覆盖
一般雇员 所有受雇员工（全职、兼职、临时工） ✅ 是
家庭佣工 包括外籍家庭佣工 ✅ 是
学徒 所有学徒 ✅ 是
自雇人士 独立承包商 ❌ 否
公务员 政府雇员 ❌ 否（由政府保障）

1.4 强制购买要求

⚠️ 雇主必须注意

  • 必须购买：所有雇主必须为所有雇员购买 ECI
  • 生效时间：从雇员入职第一天起必须生效
  • 保额要求：保险金额不少于《雇员补偿条例》规定的补偿金额
  • 持续覆盖：保险必须在整个雇佣期间保持有效

💰 二、ECI补偿范围与标准

2.1 补偿范围

📋 可获补偿的情况

  • 工伤事故：在工作时间和工作场所内发生的意外事故
  • 上下班途中：往返工作地点的合理路线上的意外事故
  • 职业病：因工作环境或工作性质导致的疾病
  • 工作相关活动：雇主指派的工作相关活动（如出差、培训）

2.2 补偿类型

补偿类型 适用情况 补偿标准
医疗费用 工伤治疗 实际医疗费用，无上限
工伤病假津贴 因工受伤不能工作 每月假期的80%，最高HKD 24,000
永久丧失能力补助 永久性伤残 根据丧失程度，最高96个月收入
死亡赔偿 因工死亡 84个月收入，最低HKD 350,000，最高HKD 4,224,000
丧葬费 因工死亡 HKD 30,000

2.3 不在补偿范围内的情况

❌ 不予补偿的情况

  • 雇员故意行为：雇员故意自伤或因犯罪行为导致的伤害
  • 非工作相关：与工作无关的个人意外事故
  • 酗酒或吸毒：因酗酒或非法药物使用导致的事故

🔄 三、ECI购买流程与管理

3.1 保险购买流程

📝 购买步骤

  1. 评估需求：根据雇员人数和行业风险确定保额
  2. 选择保险公司：从多家保险公司中选择合适的保险方案
  3. 提供信息：向保险公司提供公司信息和雇员详情
  4. 签订保单：确认保单条款和保险金额，签订保单
  5. 保存保单：妥善保存保险单据，并在工作场所展示
  6. 定期续保：在保单到期前及时续保

3.2 保险金额计算

💡 保险金额计算方法

  • 最低要求：保险金额不少于《雇员补偿条例》规定的最高补偿金额
  • 死亡赔偿：84个月收入（最高HKD 4,224,000）
  • 永久丧失能力：96个月收入
  • 建议保额：建议购买超过最低要求的保险，以充分保障雇员权益

3.3 保单管理要求

📋 雇主必须做到

  • 展示保单：在显眼位置展示保险证明副本
  • 更新信息：雇员人数或工作性质变更时，及时更新保险信息
  • 按时续保：确保保险持续有效，无空档期
  • 记录保存：保存保险单据至少7年

⚠️ 四、工伤事故处理流程

4.1 雇主责任与流程

🚨 工伤事故处理步骤

  1. 立即救助：第一时间提供医疗救助，确保雇员安全
  2. 报告事故：在事故发生后24小时内通知劳工处
  3. 提交表格：提交Form 2（雇主报告表格）给劳工处
  4. 联系保险公司：及时通知保险公司，启动索赔流程
  5. 记录事故：详细记录事故经过、原因和处理措施
  6. 跟进伤情：持续关注雇员的康复情况
  7. 支付补偿：按照法规和保险条款支付相应的补偿

4.2 关键时限要求

时限 行动 责任方
24小时内 通知劳工处事故发生 雇主
7天内 提交Form 2表格 雇主
14天内 提交Form 2A表格（如需） 雇主/雇员
6个月内 提交工伤补偿申请 雇员

4.3 常用表格说明

📄 重要表格

  • Form 2：雇主报告工伤事故表格（必须提交）
  • Form 2A：雇员工伤补偿申请表格
  • Form 5：医疗证明表格
  • Form 7：评估委员会评估报告

⚖️ 五、违规处罚与法律责任

5.1 违规类型与处罚

违规行为 处罚 后果
未购买ECI 罚款HKD 100,000 + 监禁2年 刑事定罪 + 自行承担所有补偿责任
未展示保险证明 罚款HKD 50,000 + 监禁6个月 行政处罚
未按时报告事故 罚款HKD 50,000 行政处罚
未支付补偿 罚款HKD 250,000 + 监禁3年 刑事定罪 + 双倍赔偿

5.2 民事责任

⚠️ 雇主的民事责任

  • 疏忽责任：如果事故因雇主疏忽导致，雇员可另行提起民事诉讼
  • 赔偿范围：包括痛苦与创伤、未来收入损失等额外赔偿
  • 保险覆盖：部分ECI保单包含民事责任保障

⚙️ 六、SAP系统中的ECI管理

6.1 SAP HCM/Payroll中的ECI配置

🔧 关键配置步骤

  1. 设置保险类型：在SAP中创建ECI保险类型（使用Table V_T7XXZ_INSURANCE）
  2. 维护保险信息：维护保险公司和保单信息（使用Table V_T7XXZ_INSURER）
  3. 配置工资项目：设置ECI相关工资项目（如雇主供款、保险费）
  4. 设定扣款规则：配置保险费的扣款和计算规则
  5. 维护雇员保险信息：在IT0088（保险信息）中维护雇员的ECI详情

6.2 关键信息类型

信息类型 用途 关键字段
IT0088 保险信息 保险公司、保单号、生效日期
IT0014 经常性支付/扣除 保险费金额、扣款码
IT0008 基本工资 用于计算保险金额的收入

6.3 工资核算配置

💰 工资核算规则

  • 保险费计算：根据雇员人数、行业风险、保额计算保险费
  • 扣款方式：保险费通常作为雇主的成本，不从雇员工资扣除
  • 报告生成：生成保险报告，用于向保险公司申报
  • 审计追踪：记录所有保险相关的计算和支付

📊 七、实践案例分析

案例1：建筑工人工伤事故

背景：某建筑公司一名工人在工地施工时，从脚手架上跌落，导致腿部骨折。

处理流程：

  • 立即送医治疗，医疗费用由保险公司承担
  • 24小时内通知劳工处，提交Form 2
  • 工人休假3个月，获得工伤病假津贴（每月假期的80%）
  • 康复后评估为10%永久丧失能力，获得相应补偿

关键点：及时报告和医疗救助是处理工伤事故的关键。

案例2：办公室职员上下班途中车祸

背景：某公司会计在上班途中，于合理路线上遭遇车祸受伤。

处理流程：

  • 确认事故发生在合理通勤路线上
  • 提交工伤报告和医疗证明
  • 保险公司调查后确认属于工伤范围
  • 员工获得医疗费用和病假津贴

关键点：上下班途中的意外事故也可获ECI补偿，需证明在合理路线上。

案例3：职业病认定

背景：某化工厂工人长期接触有害化学物质，被诊断为职业病。

处理流程：

  • 医生出具职业病诊断证明
  • 提交Form 2A工伤补偿申请
  • 评估委员会评估丧失能力程度
  • 根据评估结果获得永久丧失能力补偿

关键点：职业病的认定需要专业的医学证明和评估。

✨ 八、ECI管理最佳实践

8.1 风险管理

🛡️ 预防措施

  • 定期培训：为员工提供安全培训，提高安全意识
  • 风险评估：定期评估工作场所的风险
  • 改善环境：投资改善工作环境，减少工伤风险
  • 购买充足保险：确保保险金额充分覆盖潜在风险

8.2 系统化管理

📋 管理建议

  • 建立档案：为每位员工建立工伤保险档案
  • 定期审查：定期审查保险保单和覆盖范围
  • 应急预案：制定工伤事故应急预案
  • 持续改进：从事故中学习，持续改进安全管理

📝 九、学习总结

🎯 今日核心要点

  • ECI是香港雇主必须为所有雇员购买的强制性保险
  • 保险依据是《雇员补偿条例》（第282章）
  • 覆盖范围包括工伤事故、上下班途中意外、职业病
  • 补偿类型包括医疗费用、病假津贴、永久丧失能力补助、死亡赔偿
  • 工伤事故需在24小时内报告劳工处，7天内提交Form 2
  • 未购买ECI将面临最高罚款10万港币和监禁2年
  • SAP系统中通过IT0088、IT0014等信息类型管理ECI
