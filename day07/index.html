⚙️ 三、PCR规则创建

3.1 PCR规则结构

🎯 PCR规则组成要素

  • 规则名称：唯一标识（如XHK6、XHK7）
  • 操作代码：执行特定操作（AMT、ADD、SUB等）
  • 操作参数：操作的目标对象（工资类型、变量）
  • 条件判断：IF-ELSE逻辑分支
  • 注释：说明规则用途（以*开头）

3.2 薪俸税PCR规则完整示例

💻 PCR规则：XHK6 (薪俸税完整计算流程)

PCR: XHK6 (香港薪俸税 - 完整计算流程)
--------------------------------------

XHK6
* ========================================
* 香港薪俸税完整计算流程
* 版本：1.0
* 日期：2026-03-25
* ========================================

* 步骤1: 计算应税收入总额
* -----------------------

* 读取基本工资（工资类型1001）
AMT= 1001
FIEL= /TOTAL_INC

* 读取奖金（工资类型1101）
AMT= 1101
ADD  /TOTAL_INC

* 读取房屋津贴（工资类型1201）
AMT= 1201
ADD  /TOTAL_INC
FIEL= /TOTAL_INC

* 步骤2: 计算免税额
* -----------------

* 读取基本免税额（工资类型3001）
AMT= 3001
FIEL= /TOTAL_ALLW

* 判断是否已婚
IF    P0002-FAMST = '1'
  * 已婚，增加已婚人士免税额
  AMT= 3002
  ADD  /TOTAL_ALLW
ENDIF

* 判断是否有子女
IF    P0002-FAMST = '2'
  * 有子女，增加子女免税额
  AMT= 3003
  ADD  /TOTAL_ALLW
ENDIF

* 判断是否供养父母
IF    P0002-FAMST = '3'
  * 供养父母，增加父母免税额
  AMT= 3004
  ADD  /TOTAL_ALLW
ENDIF
FIEL= /TOTAL_ALLW

* 步骤3: 计算可扣除项目
* ---------------------

* 读取强积金供款（工资类型2001）
AMT= 2001
FIEL= /TOTAL_DED

* 读取慈善捐款（工资类型2002）
AMT= 2002
ADD  /TOTAL_DED

* 读取居所贷款利息（工资类型2003）
AMT= 2003
ADD  /TOTAL_DED
FIEL= /TOTAL_DED

* 步骤4: 计算应评税入息
* ---------------------

* 应评税入息 = 应税收入 - 免税额 - 可扣除项目
AMT= /TOTAL_INC
SUB  /TOTAL_ALLW
SUB  /TOTAL_DED
FIEL= /STI

* 步骤5: 按比例计算（年度中期入职）
* --------------------------------

* 检查是否有按比例标志
IF    P0001-ZVDAT = '20240701'
  * 2024年7月1日入职，按比例计算
  AMT= /STI
  * 受雇比例：75.07% (274天/365天)
  MULT 0.7507
  FIEL= /STI_PRORATA

  * 按比例免税额
  AMT= /TOTAL_ALLW
  MULT 0.7507
  FIEL= /ALLW_PRORATA

  * 更新应评税入息
  AMT= /STI_PRORATA
  FIEL= /STI
ENDIF

* 步骤6: 计算累进税率税额
* -----------------------

* 初始化税额为0
ZERO= /TX_PROG

* 第1级税率：首HKD 50,000 × 2%
AMT= /STI
AMT? >0
  AMT? <=50000
    * 全部在第1级
    MULT 0.02
    FIEL= /TX_PROG
  ELSE
    * 超出第1级部分
    AMT= 50000
    MULT 0.02
    FIEL= /TX_PROG
    * 计算剩余金额
    SUB  /STI
    FIEL= /REMAIN1
  ENDIF
ENDIF

* 第2级税率：次HKD 50,000 × 6%
AMT= /REMAIN1
AMT? >0
  AMT? <=50000
    * 全部在第2级
    MULT 0.06
    ADD  /TX_PROG
    FIEL= /TX_PROG
  ELSE
    * 超出第2级部分
    AMT= 50000
    MULT 0.06
    ADD  /TX_PROG
    FIEL= /TX_PROG
    * 计算剩余金额
    SUB  /REMAIN1
    FIEL= /REMAIN2
  ENDIF
ENDIF

* 第3级税率：次HKD 50,000 × 10%
AMT= /REMAIN2
AMT? >0
  AMT? <=50000
    * 全部在第3级
    MULT 0.10
    ADD  /TX_PROG
    FIEL= /TX_PROG
  ELSE
    * 超出第3级部分
    AMT= 50000
    MULT 0.10
    ADD  /TX_PROG
    FIEL= /TX_PROG
    * 计算剩余金额
    SUB  /REMAIN2
    FIEL= /REMAIN3
  ENDIF
ENDIF

* 第4级税率：次HKD 50,000 × 14%
AMT= /REMAIN3
AMT? >0
  AMT? <=50000
    * 全部在第4级
    MULT 0.14
    ADD  /TX_PROG
    FIEL= /TX_PROG
  ELSE
    * 超出第4级部分
    AMT= 50000
    MULT 0.14
    ADD  /TX_PROG
    FIEL= /TX_PROG
    * 计算剩余金额
    SUB  /REMAIN3
    FIEL= /REMAIN4
  ENDIF
ENDIF

* 第5级税率：余下 × 17%
AMT= /REMAIN4
AMT? >0
  MULT 0.17
  ADD  /TX_PROG
  FIEL= /TX_PROG
ENDIF

* 步骤7: 计算标准税率税额
* -----------------------

* 读取应评税入息
AMT= /STI

* 计算标准税率税额 (15%)
MULT 0.15
FIEL= /TX_STD

* 步骤8: 选择较低税额
* ------------------

* 比较累进税率税额和标准税率税额
AMT= /TX_PROG
AMT? < /TX_STD
  * 累进税率较低
  AMT= /TX_PROG
  FIEL= /FINAL_TAX
ELSE
  * 标准税率较低
  AMT= /TX_STD
  FIEL= /FINAL_TAX
ENDIF

* 步骤9: 写入薪俸税工资类型
* -------------------------

* 将最终税额写入工资类型4001
AMT= /FINAL_TAX
AMT= 4001

* 清除临时变量
ELIM= /TOTAL_INC
ELIM= /TOTAL_ALLW
ELIM= /TOTAL_DED
ELIM= /STI
ELIM= /STI_PRORATA
ELIM= /ALLW_PRORATA
ELIM= /TX_PROG
ELIM= /TX_STD
ELIM= /FINAL_TAX
ELIM= /REMAIN1
ELIM= /REMAIN2
ELIM= /REMAIN3
ELIM= /REMAIN4

❓ 六、常见配置问题

6.1 FAQ 1: 如何处理年度中期入职的免税额按比例计算？

💡 解决方案

  • 计算受雇比例：根据入职日期到课税年度结束的天数计算
  • 按比例免税额：基本免税额 × 受雇比例
  • PCR规则处理：在PCR中判断入职日期，自动按比例计算
  • 示例：7月1日入职，受雇比例 = 274/365 = 75.07%

6.2 FAQ 2: 薪俸税计算结果与手工计算不一致怎么办？

💡 排查步骤

  • 步骤1：检查PCR规则逻辑是否正确
  • 步骤2：检查工资类型金额是否正确输入
  • 步骤3：检查税表税率是否更新到最新
  • 步骤4：检查免税额是否正确计算
  • 步骤5：使用调试模式（Debug模式）逐步检查计算过程

6.3 FAQ 3: 如何在SAP中配置股票期权收入？

💡 配置方法

  • 创建工资类型：1301（股票期权收入）
  • 配置PCR规则：在收入计算阶段读取股票期权收入
  • 征税时机：在行权日（Exercise Date）计入应税收入
  • 计算公式：（行权日价格 - 行权价格）× 行权股数
  • 税务处理：作为一次性收入计入年度应税收入

6.4 FAQ 4: 如何处理月度预缴税金的计算？

💡 配置方法

  • 创建工资类型：4002（月度预缴税金）
  • 预缴比例：前6个月75%，后6个月25%
  • PCR规则：XHK8（月度预缴计算）
  • 结算机制：年度申报时多退少补
  • 示例：上年税额HKD 60,000，前6个月每月预缴HKD 3,750
