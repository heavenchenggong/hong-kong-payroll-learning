## 📋 项目简介

这是为2026年度香港Payroll客户咨询和方案设计准备的学习网站，采用"每天学一个专题，学透每个知识点"的方式，30天全面掌握香港Payroll。

## 🚀 快速开始

### 本地预览
```bash
# 克隆仓库
git clone https://github.com/heavenchenggong/hong-kong-payroll-learning.git

# 进入项目目录
cd hong-kong-payroll-learning

# 使用任何HTTP服务器预览
python -m http.server 8000
# 或
npx serve
```

然后访问 http://localhost:8000
